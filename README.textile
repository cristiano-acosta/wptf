h1. WPTF (WordPress Theme Framework)

WordPress 3.3.1 Theme designed to be a meta framework that allows you to include new taxonomies, custon post types, and related fields allowing use as CMS.
Using 960gs grid css columns with 12,16,24 and to adapt the adaptive gs 960 on various platforms.
Using latest version of jQuery, jquery ui and other plugins for editing a variety of effects.
Implementing a framework for css3 html5 standard and construction of animations.
Compatible with Adobe Edge 4.
